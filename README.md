### 3-2
入力した２つの自然数から、ユークリッドの互除法を使って最大公約数を求めて表示するプログラムを作成する。

#### 最大公約数を求めるアルゴリズム
2つの自然数aとbを入力する。ただし a ≧ b
1. aをbで割った余りcを計算する
2. cが0ならば(割り切れたら)、最大公約数はbである
3. cが0でなければ、bを新しいa、cを新しいbとする
4. 1に戻る
